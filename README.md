# BST_SLAM
BST SLAM (Blood Sweat & Tears/BeaST Simultaneous Localization And Mapping)

# WIP/POC
Work In Progress/Proof Of Concept
Don't expect this to be production ready, quite yet. It's close though. 
It's also not a real SLAM yet, because right now it's just the odometry, so there's no mapping yet. 
I do plan to add mapping, and I have some code I've been working on for that, 
but for now I set the name to what it will be ahead of time.

# Docs are also a WIP
This repo is intended to be tiny, so clean code isn't a concern, 
but while I don't have many comments, I do plan to add more to this readme. 
For now, add your sensor code to the subscriber, and yes that will take some time to setup 
(But hey, at least the system is self-contained so you don't need ROS)
And if you're really adventurous, look at the publisher code. 
The publisher "publishes" the pose for your subscriber to "subscribe to" (read: see/use).
